Tagu sarasas

  • p - paragraph - Paragrafas skirtas rašyti pastraipai
  • h1-h6 - Heading1-Heading6 - Antraštės skirtos rašyti pavadinimams ir tarpusavyje skiriasi teksto dydžiu
  • img - Image - Paveiksliukas
  • a - Hypertext Anchor (link) - Nuoroda
  • hr - Thematic Break (horizontal line) - Sukuria eilutės lūžį su linija
  • br - Line Break - Eilutės lūžis
  • ul - Unordered List - Nerikiuotas sąrašas
  • ol - Ordered List - Rikiuotas sąrašas
  • li - List Item - Sąrašo elementas (skirtas rikiuotam arba nerikiuotam sąrašui)
  • meta - Skirtas nurodyti puslapio meta duomenis
  • title - Skirtas nurodyti puslapio tab'o pavadinimą

Atributų sąrašas

  • lang - Language - Nurodo kokia kalba vyrauja tame tag'e (naudinga automatiniams skaitytuvams)
  • charset - Character set - Simbolių rinkinys nurodo kokią simbolių biblioteką naudoti
  • src - Source - Šaltinis nurodantis failą
  • alt - Alternative text - Alternatyvus tekstas paveiksliuko neužkrovimo atveju
  • width - Nurodo elemento plotį
  • height - Nurodo elemento aukštį
  • href - Hyper link reference - Nuoroda į puslapį arba failą
  • target - Nurodo kur atidaryti nuorodą

Klaviatūros trumpinių sąrašas

  • CTRL S - Išsaugoja failą
  • CTRL C - Kopijuoja pažymėta informaciją
  • CTRL X - Iškerpa pažymėtą informaciją
  • CTRL V - Įklijuoja kopijuotą/iškirptą informaciją
  • CTRL Z - Atšaukia paskutinį veiksmą/us
  • CTRL F - Atidaro paieškos sistemą
  • TAB - Tabuliuoja pažymėtą kodą į dešinę
  • SHIFT TAB - Tabuliuoja pažymėtą kodą į kairę
  • CTRL A - Pažymi viską
  • CTRL / - Komentuoja pažymėtas eilutes
  • CTRL Y | CTRL SHIFT Z - Atšaukia CTRL Z veiksmą/us
  • ALT Z - Word Wrap
  • CTRL R - Duoda pasirinkti ką atidaryti iš failų/aplankalų istorijos
  • alt + skaičiųKombinacija - Parašo skaičiųKombinacijai priskirtą simbolį
  • CTRL SHIFT L - Pažymi visus vienodus žodžius/sakinius, taip pat galima juos keisti
  • CTRL SHIFT K - Ištrina pažymėtą/as eilutę/es
  • CTRL SHIFT i/j | F12 - Atidaro Developer Tools (naršyklėje)
  • CTRL D - Pažymi tokį patį žodį einantį toliau kode
  • END - Nukels kursorių į eilutės galą
  • CTRL END - Nukels kursorių į kodo galą
  • HM (home) - Nukels kursorių į eilutės pradžią
  • CTRL HM (home) - Nukels kursorių į kodo pradžią
  • CTRL ALT N - Paleidžia executable kodą
  • SHIFT ALT arrowUP/DOWN - Nukopijuoja pažymėtą/as eilutę/es aukštyn/žemyn
